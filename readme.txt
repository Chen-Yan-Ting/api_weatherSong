WeatherSong

usage：
用firefox開啟test.html，允許定位功能，網頁就會自己跑了~
根據不同天氣播放不同的歌，遇到喜歡的歌手可以搜尋。

1.
使用了yahoo weather api, spotify api，使用了firefox內建的定位系統
，讓yahoo weather api即時顯示天氣，在利用yahoo api 回傳的code來決
定要播放的歌曲類型，串流歌曲使用spotify。

2.
使用notepadd++, firefox, node.js(spotify 登入用)，需要的基礎為html,
javascript, css，安裝基本上沒有特別困難的地方。

3.
javascript,css,node.js -> W3C school
spotify的Api使用基本上以官網為主，stack overflow為輔
零基礎的人建議先把基本知識看完再去看官網的範例，會比較好上手，

使用spotify我覺得是一個滿錯誤的決定，當初會選擇spotify是因為他是
音樂播放軟體的龍頭，然而，spotify api是一個滿新的東西(2011年)，所以
網路上的資源基本上以官網為主，可想而知，在學習的時候會一直撞牆，尤其是
對第一次接觸javascript的新手會很痛苦。
